# GPTgps 快速使用

## 这是什么

GPTgps 是一个面向长对话场景的 AI 聊天导航扩展。

它的目标不是替代原网页，而是在原网页右侧增加一套更适合长任务的导航层，帮助你：

- 快速回到之前某条提问
- 把长对话手动拆成多个 segment
- 用 mark / checkpoint 标记正文中的关键位置
- pin 重点节点
- 给节点写 note
- 用 AI 自动生成段落或条目摘要

## 当前支持的网站

- ChatGPT
- Gemini
- 千问 / 通义 / Qianwen
- 豆包
- Claude

## 核心功能

- Segment 分段导航
- Prompt / Mark 节点列表
- 搜索、Pin、Split、Rename、Note
- 正文位置锚定的 Mark / Checkpoint
- 右侧简化跳转导航
- AI 段摘要与条目摘要
- 导出 / 导入当前会话导航状态
- 诊断面板 Diag

## 安装方法

### Chrome / Edge

1. 打开 `chrome://extensions`
2. 开启右上角“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择项目目录中的 `dist_mv3`

### Firefox

1. 打开 `about:debugging#/runtime/this-firefox`
2. 点击“加载临时附加组件”
3. 选择 `dist_mv2/manifest.json`

## 基本使用

1. 打开任意已支持的 AI 聊天网页
2. 点击 GPTgps 悬浮按钮展开侧栏
3. 在侧栏中：
   - 搜索 prompt / note
   - 点击条目跳转到正文
   - 对某条 prompt 执行 pin / split / note
   - 对某个 segment 执行 rename / regen / mark
4. 如需 AI 摘要，打开 AI 设置面板，填写 provider / api key / model / base url 后保存

## Tools 说明

- `Diag`
  - 查看当前平台、会话 ID、DOM 扫描数量、parsed prompt 数量、最终列表数量
  - 适合排查“为什么没有识别到 prompt”或“为什么跳转不对”
- `Export`
  - 导出当前会话的 GPTgps 导航状态为 JSON
- `Import`
  - 导入之前导出的 JSON，恢复当前会话的导航状态

## 当前已知限制

- 连续多张仅图片消息，后续计划做成“一个可展开节点”而不是多条重复条目

## 推荐使用方式

- 日常主用：ChatGPT / Gemini / 千问
- 轻量验证：Claude
- 日常可用：豆包（图片-only 消息会以占位 prompt 形式保留）
