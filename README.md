# GPTgps

**Language / 语言**

- [简体中文](./README.zh-CN.md)
- [English](./README.en.md)

GPTgps is a multi-site AI chat navigator for long conversations.  
GPTgps 是一个面向长对话场景的多站点 AI 聊天导航扩展。

It helps you find earlier prompts, split large chats into segments, add markers and notes, and generate AI summaries from saved context.  
它帮助你找回 earlier prompts、整理长对话分段、添加标记与注释，并基于上下文生成 AI 摘要。
