GPTgps is a browser extension for navigating long AI conversations.

- Segment long chats into prompt groups
- Search, pin, note, and jump between key turns
- Save local history and reopen useful threads
- Generate AI summaries from saved context

Most data stays in your browser's local storage. GPTgps does not run its own server for storing your conversations.
